Clases universitarias

Clases particulares universitarias con diagnóstico, plan y seguimiento

Para estudiantes que necesitan ordenar una asignatura técnica, preparar un examen o recuperar criterio después de semanas de bloqueo.

Materias

  • Clases particulares de Matemáticas universitarias: Clases de matemáticas universitarias para Cálculo, Álgebra, Ecuaciones Diferenciales y asignaturas cuantitativas con método de problemas.
  • Clases particulares de Estadística universitaria: Clases de estadística universitaria para probabilidad, inferencia, contrastes, regresión y análisis de datos con interpretación rigurosa.
  • Clases particulares de Programación universitaria: Clases de programación universitaria desde cero para Python, SQL, fundamentos, proyectos y asignaturas de informática.
  • Clases particulares de Física para Ingeniería: Clases de física para ingeniería con foco en problemas, unidades, modelos, razonamiento y preparación de exámenes.
  • Clases particulares de Economía y ADE: Apoyo en economía, ADE, matemáticas financieras, estadística, microeconomía y asignaturas cuantitativas de empresa.
  • Clases particulares de Química: Clases de química para bachillerato, universidad inicial y preparación de exámenes con razonamiento, ejercicios y seguimiento.
  • Clases particulares para Selectividad y PAU: Preparación de Selectividad y PAU con calendario, simulacros, bloques de estudio y seguimiento hasta el examen.
  • Clases particulares para GCSE, IGCSE e IB: Preparación personalizada para GCSE, IGCSE e IB en asignaturas cuantitativas con rúbrica, calendario y práctica guiada.
